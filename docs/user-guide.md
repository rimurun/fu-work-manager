## 1. ログイン

### アクセス方法

ブラウザで FU Manager の URL にアクセスしてください。

### ログイン手順

1. **ユーザー名**を入力
2. **パスワード**を入力
3. 「ログイン」ボタンをクリック

ログインに失敗すると「ユーザー名またはパスワードが正しくありません」と表示されます。
正しい情報を再入力してください。

### アカウントの種類

| 種類                | 説明                                                   |
| ------------------- | ------------------------------------------------------ |
| **管理者（admin）** | 全店舗のデータ閲覧・アップロード・アカウント管理が可能 |
| **店舗（store）**   | 割り当てられた店舗のデータ閲覧・アップロードのみ可能   |

## 2. 画面構成

ログイン後、画面は左側の**サイドバー**と右側の**メインエリア**に分かれています。

### サイドバー

上から順に：

- **FU Manager** — アプリ名
- **店舗セレクター** — 表示する店舗を切り替え（複数店舗割り当て時はドロップダウン）
- **メニュー**
  - ダッシュボード — 売上・利益などの全体統計
  - キャスト分析 — キャストごとの詳細データ
  - データアップロード — Excelファイルのアップロード
  - アカウント管理 — ユーザーの店舗割り当て（管理者のみ表示）
- **ユーザー情報・ログアウト** — 画面下部にユーザー名とログアウトボタン

### 店舗の切り替え

サイドバーの店舗名をクリックするとドロップダウンが開きます。
表示したい店舗を選択すると、ダッシュボードやキャスト分析のデータがその店舗に切り替わります。

## 3. ダッシュボード

月別の店舗データを一目で把握するための画面です。

### 年月の選択

画面右上の年月セレクターで表示するデータの期間を切り替えます。

- ドロップダウンで**年**を選択
- ボタンで**月**を選択（紫色がハイライトされている月が現在選択中）

### データがない場合

「データがありません」と表示されます。
左メニューの「データアップロード」からExcelファイルをアップロードしてください。

### アラート

画面上部にカラフルなバッジで重要な情報が表示されます：

- **オレンジ** — 注意事項（売上大幅減、媒体不明が多いなど）
- **緑** — 好調指標（売上前月比+10%超など）
- **水色** — 情報（稼働率が低い時間帯、高需要時間帯など）

### 統計カード（上部4枚）

| カード             | 内容                 | 見方                                |
| ------------------ | -------------------- | ----------------------------------- |
| **月間売上**       | 当月の合計売上       | 前月比の増減%も表示（緑=増、赤=減） |
| **純利益**         | 当月の合計純利益     | 利益率も表示                        |
| **総客数**         | 当月の合計客数       | 前月比の増減%も表示                 |
| **新規・会員比率** | 新規客と会員客の割合 | ピンク=新規、水色=会員              |

### 売上・純利益推移（折れ線グラフ）

日別の売上（紫）と純利益（緑）の推移です。
特定の日にマウスを合わせると詳細な金額が表示されます。

### 曜日別平均売上（棒グラフ）

曜日ごとの平均売上額です。

- **紫色** = 金・土・日（週末）
- **灰色** = 月〜木（平日）

週末の売上が平日より高ければ正常です。

### 媒体別売上（円グラフ + リスト）

お客さんがどの媒体（サイト）から来たかの内訳です。

- 各媒体の売上金額、人数、割合が表示されます
- **「不明」が20%を超えると赤色で警告**が出ます → 媒体確認を徹底してください

### 時間帯別分析 & シフト提案

時間帯ごとの稼働率と出勤数のグラフです。

- **緑色のバー** = 需要が高い（稼働率33%超）→ キャストを手厚くしたい時間帯
- **黄色のバー** = 適正範囲
- **赤色のバー** = 余裕あり（稼働率20%以下）→ キャスト過多の可能性
- **水色の線** = 出勤キャスト数
- **緑の点線（目標28%）** = 稼働率の目標ライン

右側のパネルに「需要が高い時間帯」「余裕がある時間帯」のまとめが表示されます。

### コース別売上（横棒グラフ）

コース（60分、80分など）ごとの売上です。
売上の大きいコースから順に表示されます。

### データの削除（管理者のみ）

ダッシュボードのタイトル横にあるゴミ箱アイコンで、
選択中の年月のデータを削除できます（確認ダイアログあり）。

## 4. キャスト分析

キャストごとのパフォーマンスを詳しく分析する画面です。

### サマリーカード（上部6枚）

| カード     | 内容                              |
| ---------- | --------------------------------- |
| 総キャスト | 出勤したキャスト数                |
| 平均売上   | キャスト全体の平均売上            |
| 平均本指名 | 本指名の平均件数                  |
| エース     | 総合評価「エース」のキャスト数    |
| 要注意     | 「勤怠△」「育成」判定のキャスト数 |
| 離脱リスク | リスク「高」「中」のキャスト数    |

### アラートバッジ

クリックすると該当キャストがハイライトされます：

- **赤「離脱リスク高」** — 辞める可能性が高いキャスト
- **オレンジ「離脱リスク注意」** — 注意が必要なキャスト
- **水色「本指名転換の余地あり」** — 写真指名が多く本指名に誘導できそうなキャスト
- **緑「本指名率70%超＆高リピート」** — 優秀なキャスト
- **オレンジ「売上あるがリピート率低」** — 売上はあるがリピートが弱いキャスト

### 離脱リスク予測

キャストの離脱リスクをカードで表示します。

判定基準：

- 売上の大幅減少（前月比-20%超）
- 欠勤率の急上昇（+15ポイント以上）
- 非常に高い欠勤率（50%超）
- 低稼働率

リスクレベル：

- **高（赤）** — 複数の強い警告シグナル
- **中（オレンジ）** — 明確な警告サイン
- **低（黄）** — 軽度の懸念

### キャスト一覧テーブル

ヘッダーをクリックするとその項目でソートできます。

| 列       | 説明                           | 色の意味                          |
| -------- | ------------------------------ | --------------------------------- |
| キャスト | 名前と順位。↑=前月比UP、↓=DOWN | —                                 |
| 売上     | 当月の個人売上                 | —                                 |
| 本指名   | 本指名の件数                   | —                                 |
| 写真     | 写真指名の件数                 | —                                 |
| リピ率   | リピート率                     | 緑=30%以上、黄=20-30%、赤=20%未満 |
| 延長率   | 延長率                         | 紫=20%以上                        |
| 稼働率   | 稼働率                         | 緑=40%以上、黄=25-40%、灰=25%未満 |
| 欠勤率   | 欠勤率                         | 緑=10%以下、黄=10-20%、赤=20%超   |
| タイプ   | キャストの傾向分類             | 下記参照                          |
| 判定     | 総合評価                       | 下記参照                          |

**タイプの分類：**
| タイプ | 条件 |
|--------|------|
| リピ型 | 本指名率70%以上 & リピート率25%以上 |
| 新規型 | 写真指名 > 本指名 |
| 効率型 | 稼働率40%以上 |
| バランス | 上記に該当しない |

**判定の分類：**
| 判定 | 意味 |
|------|------|
| エース | トップパフォーマー（総合スコア高） |
| 安定 | 安定した実績 |
| 成長中 | 伸びしろがある |
| 勤怠△ | 欠勤率が高い |
| 育成 | 本指名率が低く育成が必要 |
| 様子見 | 判断保留 |

### 本指名率ランキング（横棒グラフ）

本指名＋写真指名が5件以上のキャストを対象に、本指名率（本指名÷総指名）でランキング表示。

- **ピンク** = 70%以上（優秀）
- **紫** = 50-70%
- **灰色** = 50%未満

### キャスト詳細分析（レーダーチャート）

テーブルでキャストをクリックすると、右側にレーダーチャートが表示されます。
6つの指標（売上、本指名、リピート率、延長率、稼働率、勤怠）をバランスで可視化します。

## 5. データアップロード

Excelファイルをアップロードして、ダッシュボードとキャスト分析のデータを更新します。

### アップロード手順

1. サイドバーで**アップロードしたい店舗**が選択されていることを確認
2. 画面上部で**年**と**月**を選択（データがどの期間のものかを指定）
3. Excelファイル（.xlsx または .xls）をドラッグ＆ドロップ、または「ファイルを選択」をクリック
4. アップロード完了後、ダッシュボードで確認

### 注意事項

- **対応形式**: `.xlsx`, `.xls` のみ
- **同じ年月で再アップロード**すると既存データが上書きされます
- アップロード中はステータスが表示されます（スピナー → 緑チェック=成功、赤=エラー）

## 6. アカウント管理（管理者のみ）

管理者アカウントでのみ表示されるメニューです。
各アカウントがアクセスできる店舗を設定します。

### 操作方法

1. 各ユーザーの行に表示されている**店舗ボタン**をクリック
   - **紫色** = アクセス可能（選択中）
   - **灰色** = アクセス不可
2. 必要な店舗をON/OFFしたら「**保存**」ボタンをクリック
3. 「保存しました」と表示されれば完了

### 管理者アカウント

管理者は全店舗に自動でアクセスできるため、店舗の割り当て設定はありません。

## よくある質問

### Q: ダッシュボードに「データがありません」と表示される

**A:** 選択中の年月にデータがアップロードされていません。
「データアップロード」ページから対象のExcelファイルをアップロードしてください。

### Q: 店舗を切り替えたい

**A:** サイドバーの店舗名をクリックして、表示したい店舗を選んでください。

### Q: 前月のデータと比較したい

**A:** ダッシュボードの統計カードに前月比（%）が自動で表示されます。
前月のデータもアップロードされている必要があります。

### Q: アップロードしたデータを削除したい

**A:** 管理者のみ、ダッシュボードのタイトル横のゴミ箱アイコンから削除できます。

### Q: ログアウトしたい

**A:** サイドバー下部のログアウトアイコン（右矢印）をクリックしてください。
